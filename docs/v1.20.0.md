# Autoforge v1.20.0 版本说明

> 发布日期：2026-07-22 · 应用版本 `1.20.0` · 脚本规范 `autoforge 1.0`

Autoforge 是本机自动化脚本锻造与管理桌面应用。v1.20.0 交付两项能力：**无限嵌套分类树**（含侧栏底栏压缩），以及**单脚本多实例批量运行**（最多 5 个预设槽、同脚本并发上限 5）。

完整功能基线见 [v1.9.0](./v1.9.0.md)；近期 Hub 相关见 [v1.15.0](./v1.15.0.md)、[v1.19.0](./v1.19.0.md)。本文档仅描述 v1.20.0 新增与变更。

设计规格：

- [无限树形分类与侧栏底栏](./superpowers/specs/2026-07-21-nested-category-tree-design.md)
- [单脚本多实例批量运行](./superpowers/specs/2026-07-22-script-multi-instance-batch-design.md)

## 相对 v1.19.0 的变化

| 领域 | 变更 |
|------|------|
| **分类树** | `categories.parent_id` 邻接表；侧栏 / 管理弹窗 / 选择器改为树形；筛选与计数含全部子孙 |
| **侧栏底栏** | 「上传脚本」压缩；Hub / 指南 / 历史 / 设置改为一排彩色图标 |
| **多实例批量** | 每脚本最多 5 个实例槽（名称 + 环境 + 配置 + 参数 + 无头）；勾选并行启动；单次 ▶ 保留 |
| **并发与日志** | 同脚本 running 总数 ≤ 5；日志按 session 分页签，标题优先显示实例名 |
| **数据迁移** | v4 `parent_id`、v5 `instance_slots`；启动时确保列存在（兼容版本号已记但列缺失） |

## 无限嵌套分类

### 行为摘要

- 任意深度树；任意节点可挂脚本或子分类（含内置分类下挂子节点）
- 内置分类自身仍为顶层（`parentId` 为空）
- 点选分类 → 列表过滤该节点及其全部子孙
- 右键：新建子分类 / 重命名 / 删除（有子节点或直接挂靠脚本时禁止删除）
- **不支持**拖拽改层级（本期非目标）

### 模块

| 模块 | 职责 |
|------|------|
| 迁移 `004-category-parent-id` | `categories.parent_id` |
| `shared/category-tree.ts` | 建树、子孙收集、环检测、递归计数 |
| `CategoryTreeList` / `CategoryTreeSelect` / `CategoryManagerModal` | 侧栏树、选择器、管理弹窗 |
| `Sidebar.vue` | 树形导航 + 压缩底栏 |

## 单脚本多实例批量运行

### 实例槽

每个槽包含：

| 字段 | 说明 |
|------|------|
| `name` | 显示名（日志页签优先使用） |
| `envId` | 运行环境 Profile |
| `config` | 本实例环境变量（对应详情「配置」），**不写回** `configByEnv` |
| `params` | 本实例运行参数，**不写回** `paramsByEnv` |
| `browser.headless` | 可选无头覆盖 |

上限：**最多 5 个槽**；同脚本 **running 会话（单次 + 批量）合计 ≤ 5**，超出整批拒绝。

### 入口与 UI

- 详情面板：保留 ▶ 单次运行；旁侧「批量」打开面板；运行中显示 `x/5`
- 卡片菜单：「批量运行…」；未满 5 时可继续启动
- 批量面板：槽列表勾选 / 编辑 / 删除；编辑区以 Tab 区分「环境变量」与「运行参数」；「启动所选」「停止全部」

### 运行与日志

- `startBatch(scriptId, slotIds)` 预检后按槽启动；每槽一个 `RunSession`（含 `instanceSlotId` / `instanceName`）
- 底部执行日志为**每个实例一个页签**，互不混写
- `stop(sessionId)` 停单个；批量面板「停止全部」调用 `stopByScript`

### 模块

| 模块 | 职责 |
|------|------|
| 迁移 `005-instance-slots` | `script_preferences.instance_slots` JSON |
| `shared/instance-slots.ts` | 规范化、上限校验；IPC 前打成纯对象 |
| `script-runner` | 并发计数、`persistParams` / `configOverride` / `browserOverride`、`startBatch` |
| `BatchRunPanel.vue` | 批量 UI |
