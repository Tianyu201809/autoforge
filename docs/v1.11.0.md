# Autoforge v1.11.0 版本说明

> 发布日期：2026-07-04 · 应用版本 `1.11.0` · 脚本规范 `autoforge 1.0`

Autoforge 是运行在 Windows 上的 Electron 桌面应用，作为**本机自动化脚本锻造与管理平台**。v1.11.0 在 v1.10.0 能力基线上，重点优化**设置、执行历史、开发指南**等辅助功能的交互形态，并改进侧边栏与详情面板的布局体验。

完整功能清单见 [v1.9.0 版本说明](./v1.9.0.md)；数据隔离与运行确认见 [v1.10.0 版本说明](./v1.10.0.md)。本文档仅描述 v1.11.0 新增与变更。

---

## 相对 v1.10.0 的变化

| 领域 | 变更 |
|------|------|
| **功能弹窗** | 设置、执行历史、开发指南改为居中模态弹窗（`AppFeatureModal`），主工作区不再被挤占 |
| **侧边栏布局** | 分类列表可独立滚动；底部「导入脚本」按钮始终可见 |
| **详情 Tab 栏** | 面板收窄或 Tab 较多时，当前 Tab 自动滚入视口；悬停显示横向滚动条 |
| **脚本创建 Skill** | `autoforge-script-create` 流程与 reference 速查表优化 |

---

## 功能弹窗（AppFeatureModal）

v1.11.0 起，以下入口以**居中模态弹窗**呈现，而非嵌入主布局的侧栏面板：

| 入口 | 位置 | 说明 |
|------|------|------|
| **设置** | 标题栏齿轮图标 | 环境 Profile、Python / 浏览器路径、全局依赖、窗口模式、皮肤 |
| **执行历史** | 侧边栏导航 | 按日汇总、搜索与状态筛选；点击条目仍可查看详情 |
| **开发指南** | 标题栏 | 示例导入、规范下载、脚本创建指引 |

**交互：**

- 点击遮罩或按 `Esc` 关闭弹窗
- 三个弹窗互斥：打开其中一个会自动关闭其他
- 设置面板改为**懒加载**：首次打开时才拉取配置，减少启动开销

主内容区（脚本列表 + 详情面板 + 终端）在弹窗打开时保持可见，便于对照配置或历史记录操作脚本。

---

## 侧边栏分类滚动

自定义分类较多时，分类区域在侧边栏中部**独立纵向滚动**，导航区与底部「导入脚本」按钮固定不动，避免按钮被挤出可视区。

---

## 详情面板 Tab 栏

详情面板 Tab（详情 / 配置 / 定时 / 编辑等）在面板宽度较窄或 Tab 较多时：

- 支持横向滚动；鼠标悬停 Tab 栏时显示细滚动条
- 切换 Tab、调整面板宽度或窗口缩放后，**当前 Tab 自动滚入视口**
- 鼠标滚轮在 Tab 栏上可横向滚动

---

## 脚本创建 Skill

随安装包分发的 `skills/autoforge-script-create/` 在 v1.11.0 中更新：

- 前置确认项与创建流程 checklist 更清晰
- `reference.md` 字段速查与示例对齐当前 `script-spec.md`
- TypeScript / Python 模板说明补充

Agent 或开发者通过 `/autoforge-script-create` 创建脚本包时，可直接引用更新后的 Skill。
